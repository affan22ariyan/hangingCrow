# Shared UI Components

Production-ready, accessible component library for betting platform frontends.

## Features

- ✅ WCAG 2.2 AA compliant
- ✅ Keyboard navigable
- ✅ Dark/Light theme support
- ✅ i18n ready
- ✅ Zero external dependencies (React only)

## Usage

**Copy components into your frontend:**

```bash
cp -r packages/shared-ui/src/* frontend-user/src/components/
```

## Components

- `<Button>` - Primary, danger, success variants
- `<Input>` - Labeled text input with validation states
- `<Select>` - Dropdown with keyboard navigation
- `<Modal>` - Accessible dialog with focus trap
- `<Table>` - Responsive data table
- `<EmptyState>` - Empty state placeholder
- `<Spinner>` - Loading indicator
- `<Alert>` - Toast/banner notifications

## Theme

Design tokens in `theme.js` - copy to each frontend's `src/` folder.
